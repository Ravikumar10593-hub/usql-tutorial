# ARRAYs

## Overview

* An ordered list of values (all of the same type)
* Immutable
* Single-Dimension

## Sample data




## Creating ARRAYs

### Creating Empty arrays

```
new SqlArray<string> ( )
```

or

```
new SqlArray<string> { }
```

### Creating arrays initialized with data

```
new SqlArray<string> { "A", "B" }
```
